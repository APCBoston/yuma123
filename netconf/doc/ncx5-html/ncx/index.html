Network Configuration Extensions (NCX)

The IETF develops network management standards which include data models for configuration, monitoring, and notifications.

The Network Configuration Working Group developed a Proposed Standard called the Network Configuration Protocol (NETCONF).

However, the conceptual content layer is intentionally not part of the NETCONF protocol definition. In addition, several network management system components are needed in support of the NETCONF protocol, in order to provide a robust configuration management solution for networking devices.

NCX includes a data modeling language which fully supports the NETCONF Remote Procedure Call (RPC) layer, the standard protocol operations, and the content Layer.

NCX also defines a network management framework for the NETCONF protocol, to support the development of interoperable programmatic interfaces, which can evolve over time.
