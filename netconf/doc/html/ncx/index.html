Network Configuration Extensions (NCX)

The IETF develops network management standards which include data models for configuration, monitoring, and notifications.

The Network Configuration Working Group developed a Proposed Standard called the NETCONF protocol.

This document defines a Structure of Management Information (SMI) for the NETCONF protocol.

NCX is designed to provide critical management system components which are intentionally not part of the NETCONF protocol. It supports a network management framework which fully utilizes NETCONF protocol features, and also integrates existing SMIv2 based data types and Management Information Base (MIB) objects.

A management framework for NETCONF, which is suitable for the development of inter-operable data models which evolve over time, needs to consider the following issues:

  • NETCONF Protocol Operations
  • Data Modeling Language
  • Modular Data Models
  • Data Organization
  • Data Naming Scheme
  • NETCONF Access Control Model
  • SMIv2 Integration
